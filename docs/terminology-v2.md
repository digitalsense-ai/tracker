# Terminology
**Version: v2**

## Core Terms
- Breakout: Price moves outside a defined range
- Retest: Price returns to breakout level
- Range: High/low during setup window
- ATR: Volatility measure
- Risk: Amount willing to lose
- R:R: Risk vs reward ratio
- Stop-loss: Exit if wrong
- Take-profit: Exit when right
- Trailing stop: Moving stop

## Simple idea
Shared language for system clarity.
